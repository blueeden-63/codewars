# CodeWars Ruby Solutions

This repository contains my personal solutions to various katas from CodeWars, written in Ruby.

The goal of this project is to practice Ruby programming while solving algorithmic and logical challenges from the CodeWars platform.

## Tech Stack

* Ruby

## Repository Structure

Exercises are organized by kata difficulty level following the CodeWars ranking system.

```
Ruby/
├── 8_kyu/
└── 7_kyu/
```

Each file inside these directories contains a solution to a specific kata.

* **8 kyu** – Beginner level exercises
* **7 kyu** – Slightly more challenging problems

## Purpose

This repository serves two main purposes:

* Practicing and improving my Ruby programming skills
* Archiving my CodeWars kata solutions in an organized way

## Current Content

At the moment, the repository contains only the solutions for the exercises.

## About CodeWars

CodeWars is a platform where developers improve their skills by solving programming challenges called **katas**.
Each kata has a difficulty ranking ranging from **8 kyu (beginner)** to **1 kyu (advanced)**.

## Notes

Solutions may evolve over time as I discover better or more idiomatic Ruby approaches.
